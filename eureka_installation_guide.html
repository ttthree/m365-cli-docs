Installation Guide — Apr 1, 2026

Get up and running with Eureka in 5 minutes. Connect to M365, install the CLI, and start automating your workflow.

Install Eureka

  1. Open Windows PowerShell as Administrator

    Search for "Windows PowerShell" in the Start menu, then click Run as Administrator.

  2. Run the install command

    Paste the following command into the PowerShell window and press Enter:

    irm https://arenas.blob.core.windows.net/dists/install.ps1 | iex

Set up M365 Connector

  1. Install the browser extension

    Install the M365 Connector extension to your Edge browser from the Chrome Web Store:

    Get M365 Connector extension

  2. Complete the setup wizard

    Follow the instructions in the browser to set up the M365 Connector. You'll be prompted to log in to the following two sites — click Next a few times through the wizard:

    • M365 Copilot
    • Graph Explorer

    The trickiest part: on the Graph Explorer page, find the sign-in button shown below and click it.

    Look for the profile icon in the top-right corner of Graph Explorer

Get started

  1. Open Eureka and sign in

    Launch the Eureka app and log in with your GitHub account (alias_microsoft).

  2. Enable M365 Connector

    If you need access to M365 data (emails, Teams, calendar, files), enable the M365 Connector from the + menu in the chat input bar.

  3. Start using it

    Ask any question or have it perform tasks for you — research, writing docs, creating slides, and more.

M365 CLI bonus

Want Eureka to automate things like sending Teams messages, composing emails, and scheduling meetings on your behalf? Install the M365 CLI:

  1. Ask Eureka to install it for you

    Paste this into Eureka's chat:

    Install the m365 cli and skill following: https://github.com/ttthree/m365-cli-docs/blob/main/skills/m365-cli/SKILL.md

    Eureka will download and install the CLI, then walk you through authentication. Follow its instructions.

  2. Restart Eureka

    After the CLI and skill are installed, close and reopen Eureka so the new skill gets loaded.

  3. Try it out

    You can now ask Eureka to interact with M365 directly:

    > Say hi to Alex on Teams
> Schedule a 30min 1:1 with Sarah for 3PM tomorrow
> What emails did I get from my manager today?
> Read the agenda in this spreadsheet and schedule the meetings

Customization bonus

You can customize Eureka's behavior via Settings → Preferences → Notes. Some useful examples:

  • Append "(Sent by Eureka, might not be accurate)" to every message/email/calendar event sent on your behalf
  • Wait for approval before sending any Teams message, email, or calendar event unless explicitly told to send directly
Need help? Ask Eureka itself — it can explain its own capabilities, troubleshoot issues, and guide you through advanced configuration.
